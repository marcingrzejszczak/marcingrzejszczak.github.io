OSS

Below you can find a list of my open source projects that I’m working on / I’ve created

  • Spring Cloud - mostly I’m working on Spring Cloud Sleuth, Spring Cloud Contract and Spring Cloud Pipelines
  • JSON Assert - fluent JSON assertions
  • XML Assert - fluent XML assertions
  • Spock Subjects Collaborators Extension - @InjectMocks for Spock
  • Gradle Test Profiler - a Gradle plugin to profile your tests
  • Gradle Uptodate Plugin - Gradle plugin to check if your code is up to date.

I’ve also contributed some code to

  • Groovy - I’ve done the basis of the @Builder transform
  • Mockito - I’ve done some work on deep stubs (hopefully you shouldn’t be using them!)
  • Rest-Assured - the Async bit of Spring MVC (version 2.5.0)
  • Micro-Infra-Spring - a lot from that code is already in Spring. I’m not contributing to this repo anymore though.
  • Moco - added GMoco support
  • Accurest - Consumer Driven Contracts verifier for Java - predecessor of Spring Cloud Contract
